# DeepReelBuilder
🛠️ Инструмент для сборки глубоких интеллектуальных видео из локальных изображений, видео и JSON‑сценария. Полностью бесплатный стек: Python, MoviePy, Edge‑TTS, GitHub Actions. Без API‑ключей, без подписок.

# DeepReelBuilder – собери глубокое видео из своих материалов

**DeepReelBuilder** позволяет превратить заранее подготовленные изображения и короткие видео в полноценный 10–30‑минутный ролик с профессиональной озвучкой и субтитрами. Вам нужен только JSON‑сценарий (можно сгенерировать с помощью AI‑ассистента) и папка с визуальными файлами.

Идеально для:
- Философских и научных видео‑эссе
- Лекций и объяснительных роликов
- Художественных проектов с единой визуальной эстетикой

### 🧩 Как это работает

1. Вы получаете (или пишете сами) JSON‑сценарий с фразами и промптами.
2. Вручную создаёте изображения и видео по промптам в любом сервисе (Midjourney, Kandinsky, Photoshop, etc.).
3. Помещаете файлы в папку `assets/` с именами `scene_00.jpg`, `scene_01.mp4` и т.д.
4. Запускаете `python main.py scene_script.json` – и получаете готовое видео.

### ⚙️ Технологии

- **Python 3.11+**
- **MoviePy** – сборка видео, субтитры, композитинг
- **Edge‑TTS** – бесплатная высококачественная озвучка (русский, английский и десятки других языков)
- **Pillow** – работа с изображениями
- **FFmpeg** – кодирование видео (программное или аппаратное)
- **GitHub Actions** (опционально) – полностью облачная сборка, не нагружающая ваш компьютер

### 💸 Полная автономность и бесплатность

- Никаких API‑ключей, платных сервисов или регистраций.
- Озвучка выполняется локально или на GitHub‑раннере через бесплатный Edge‑TTS.
- При использовании GitHub Actions вы получаете современный CPU‑рендеринг, который собирает видео длительностью 20 минут за 15–30 минут.

### 🚀 Быстрый старт

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Положите файл сценария `scene_script.json` и изображения в `assets/`
4. Выполните: `python main.py scene_script.json`
5. Готовое видео: `finished_cloud_video.mp4`

### 🤖 Совместная работа с AI‑ассистентом

Для генерации JSON‑сценария на сложные темы можно использовать DeepSeek, ChatGPT, Claude или любого другого AI‑помощника.  
Просто скопируйте инструкцию из файла `PROMPT_FOR_AI.md` в новый чат и приложите свою тему – ассистент вернёт готовый сценарий с корректными промптами, совместимый с этим проектом.
